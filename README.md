# Full Stack Open

## About

This repository contains the assignments for the online course [**Full Stack Open**](https://fullstackopen.com/).

The course focuses on modern web development and covers topics such as React, Node.js, REST APIs, and testing.

## Technologies

- JavaScript
- React
- Node.js
- Express
- HTML & CSS
